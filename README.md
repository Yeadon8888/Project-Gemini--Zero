# Project Zero-Code

这是一个基于 HTML5 和 Phaser 3 引擎开发的简单无尽跑酷游戏。玩家需要控制角色跳跃躲避障碍物，尽可能获得更高的分数。

## 🎮 游戏玩法

- **目标**: 控制角色奔跑，躲避不断出现的红色障碍物。
- **得分**: 随着存活时间增加，分数会不断累积。
- **最高分**: 游戏会自动记录你的最高分（保存在本地浏览器中）。
- **失败**: 碰到障碍物则游戏结束。

## 🕹️ 操作指南

- **开始游戏**: 点击屏幕任意位置开始。
- **跳跃**: 按下 **空格键 (SPACE)** 跳跃。
- **重新开始**: 游戏结束后，按下 **空格键 (SPACE)** 重新开始。

## 🚀 如何运行

1. 克隆或下载本项目到本地。
2. 直接在浏览器中打开 `index.html` 文件即可开始游戏。
   - 推荐使用 Chrome, Firefox 或 Edge 等现代浏览器。
   - 如果遇到本地文件跨域问题（CORS），建议使用简单的本地服务器运行，例如：
     - 使用 Python: `python -m http.server`
     - 使用 VS Code 的 "Live Server" 插件。

## 🛠️ 技术栈

- **HTML5 Canvas**: 用于游戏渲染。
- **Phaser 3**: 强大的 2D 游戏框架。
- **JavaScript (ES6)**: 游戏逻辑实现。

## 📂 项目结构

- `index.html`: 游戏入口文件。
- `game.js`: 核心游戏逻辑（场景、角色、物理引擎、动画等）。
- `assets/`: 存放游戏资源（图片、音效）。
  - `player.png`: 玩家角色精灵图。
  - `jump.wav`: 跳跃音效。
  - `game-music-loop-*.mp3`: 背景音乐。

## 📝 开发说明

游戏逻辑主要包含在 `game.js` 中：
- `preload()`: 加载资源。
- `create()`: 初始化游戏对象（背景、玩家、动画、输入监听等）。
- `update()`: 每一帧的逻辑更新（运动、碰撞检测、分数更新）。
